# Сайт оконного мастера

Полноценный сайт для оконного мастера с формой обратной связи и отправкой заявок в Telegram.

## Возможности

- ✅ Современный адаптивный дизайн
- ✅ Форма обратной связи с валидацией
- ✅ Автоматическая отправка заявок в Telegram
- ✅ Описание услуг и преимуществ
- ✅ Контактная информация
- ✅ Стилизация на SCSS

## Установка и настройка

### 1. Установка зависимостей

```bash
npm install
```

### 2. Настройка Telegram бота

1. Создайте бота через [@BotFather](https://t.me/BotFather) в Telegram:
   - Откройте чат с @BotFather
   - Отправьте команду `/newbot`
   - Следуйте инструкциям и получите токен бота

2. Узнайте свой Chat ID:
   - Напишите боту [@userinfobot](https://t.me/userinfobot)
   - Он вернёт ваш Chat ID

3. Создайте файл `.env` в корне проекта:

```env
TELEGRAM_BOT_TOKEN=your_bot_token_here
TELEGRAM_CHAT_ID=your_chat_id_here
```

**Важно:** Не коммитьте файл `.env` в git! Он уже добавлен в `.gitignore`.

### 3. Запуск разработческого сервера

```bash
npm run dev
```

Сайт будет доступен по адресу: `http://localhost:3000`

### 4. Сборка для production

```bash
npm run build
```

## Настройка контактной информации

Для изменения контактной информации (телефон, адрес) отредактируйте файл:
- `app/pages/index.vue` - найдите все вхождения `+79991234567` и `г. Москва, ул. Примерная, д. 1`

## Развёртывание

Проект готов к развёртыванию на любой платформе, поддерживающей Node.js (Vercel, Netlify, Railway и т.д.).

Не забудьте добавить переменные окружения `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID` в настройках вашей платформы развёртывания!
